# 1. Имя тест-кейса

## Формат входных данных

```
1. Имя тест-кейса
   - Изменить имя на: «Новое название тест-кейса»
```

Или с альтернативой:
```
1. Имя тест-кейса
   - Изменить имя на: «Вариант 1»
   - или: «Вариант 2»
```

## Алгоритм

1. Найти строку с `Изменить имя на:`
2. Извлечь текст из кавычек `«...»`
3. Если несколько вариантов с `или:` — используй первый
4. Удали кавычки из названия
5. Вызови `testcase_patch` с параметром `name`

## MCP инструменты

```python
# Получить текущее имя
mcp__allure_http__testops_testcase_get(testcase_id=80532)

# Изменить имя
mcp__allure-http__testops_testcase_patch(
    testcase_id=80532,
    name="Новое название"
)
```

## Валидация

- Новое имя не должно быть пустым
- Максимальная длина: 255 символов (TestOps ограничение)
